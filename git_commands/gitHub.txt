Перенос на удаленный репозиторий 

* git remote add origin github_URL  ==> связывает пустой удаленный репозиторий (по ссылке) с докальным гит репозиторием 
* git remote -v ==> для получения имени удаленного репозитория + ссылка на него
* git push -u origin branch-name ==> передвет изменения из ветки локального репозитория на ветку удаленного
          -u ==> (uppstream) "связывает" локальную и удаленную ветку и последующие команды
          могут быть упрощены, только "git push"

Клонирование удаленного репозитория:           

* git clone <URL_удаленного_репозитория> ==> клонирование с удаленного репозитория на локальный.
* git fetch ==> получить последние изменения из удаленного репозитория в стэйджинг (индекс) 
* git pull origin branch-name ==> подтягивает все изменения из remote в локальный репозиторий
