

* git status               // показывает текущее состояние
* git init                 // инициализирует папку (проект)
* git log                  // выводит историю коммитов репозитория
* git log --oneline         // выводит короткую историю коммитов
* git branch               // отображает список веток в репозитории (текущая помечена звездочкой)
* git remote               // отображает список удалённых репозиториев
* git show                 // отображает информацию о коммите
* git config user.name     // отображает имя пользователя в гите
* git config user.email    // отображает электронную почту пользователя
* git diff                 // отображает разницу между версиями
