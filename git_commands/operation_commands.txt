

* q - для выхода из истории просмотров и возвращения к командной строке
* git switch name_branch - переключает на новую ветку
* git checkout name_branch - переключает на новую ветку
* git checkout -b new_branch  - создаёт новую ветку и переключает на неё
* git branch -d name_branch -  Git не удаляет историю коммитов из репозитория. Он только удаляет ссылку на ветку
